Introduction

    BC-House Interiors asset packs are collections of prefabs built to real-world scale,
	designed to help game developers design 3D levels quickly by providing collections of themed assets that are modular,
	built to real-world scale, customizable, and easy to use.
	
	Don't have the props you need?
	No door and wall colors you need?

	Please email us, we will include it in the package as soon as possible.
	Ex) Please add a blue fan!

	Note).
	-All meshes used only one material & texture. Each mesh has its own UV for each color.
	-All meshes have small UVs, so textures larger than 1px cannot be used.
